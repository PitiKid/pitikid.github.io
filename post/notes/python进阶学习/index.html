*args 与 ** keyargs

对于这个问题之前已经了解过，现在的记忆似乎是，传入一个list和一个字典，现在重新了解一下吧

  1. *args 对于这个参数，似乎是将剩下的所有变量，放在一个元组中，顺便复习一下元组中的变量是不能改变的
  2. **keyargs 对于这个参数，是将键值对作为参数传递到函数里面，变量名是字符串

对于这个，似乎有两种使用方式

生成器 和 迭代器

其中包括三大块

－ 可迭代对象(Iterable) － 迭代器(Iterator) － 迭代(Iteration)

  1. 可迭代对象 对于可迭代对象，只要对象定义了一个可以返回一个迭代器的__iter__方法，或者定义了可以支持下标索引的__getitem__方法，即为一个可迭代对象

  2. 迭代器 对于迭代器，只要对象定义了一个__next__方法，即为迭代器

  3. 迭代 对于一个包含多个元素的对象中循环遍历取出其中的元素的过程就是迭代

  4. 生成器 生成器也算是一种迭代器，不过只能循环一次，因为它们并没有把所有的值存在内存中，而是在运行时生成值。

Map，Filter，Reduce

说句实话，这三个名词，我就在之前玩Hadoop的时候见过，之后再野没见过，也没有用过，现在学习一下吧。

  1. Map map函数会将一个函数映射到一个输入列表的所有元素上。

    1
2
3

    def tmp(x):
    return x**2

# tt = map(tmp, range(1,7))
tt = map(lambda x:x**3, range(1,7))

  2. Filter Filter过滤列表中的元素，并且返回一个由所有符合要求的元素所构成的列表，Filter将列表中的元素传入 到一个函数中，若返回True 则留下来

1
2

4

def fu(x):
    if x < 0:
	return True
    else:
	return False

print(list(filter(fu, [-1, -2 ,-3 , 4])))
print(list(filter(lambda x : x<0, [-1, -2 ,-3 , 4])))  # 使用匿名函数

  1. Reduce 用传给 reduce 中的函数 function（有两个参数）先对集合中的第 1、2 个元素进行操作，得到的结果再与第三个数据用 function 函数运算，最后得到一个结果。

    python3 中 将reduce去除全局空间了，现在需要再functools中导入
